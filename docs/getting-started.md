# Quick Start

Get up and running with Skild in 2 minutes.

---

## 1. Install Skild

```bash
npm i -g skild
```

---

## 2. Install a Skill

```bash
skild install anthropics/skills/skills/pdf
```

This installs the `pdf` skill from Anthropic's official repository.

---

## 3. Verify

```bash
skild list
```

You should see the installed skill listed.
