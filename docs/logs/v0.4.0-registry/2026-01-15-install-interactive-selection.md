# 2026-01-15 install 交互选择：Skill 树 + 平台多选

## 背景 / 问题

- 多 skill 仓库仅支持全选安装，用户无法精细选择
- 未指定平台时缺少引导，用户需要记命令

## 决策

- 多 skill 源进入交互选择：默认全选，可按目录/节点取消
- 未指定平台时弹出平台多选：默认全选
- 选完 skills 后打印已选清单

## 变更内容

- 用户可见变化：`skild install` 支持交互式选择 skills 与平台
- 关键实现点：在 CLI install 命令中加入树形多选与平台多选逻辑
- 文档更新：说明交互式选择行为

## 验证（怎么确认符合预期）

```bash
pnpm build
pnpm lint
pnpm typecheck
```

验收点：

- 多 skill 源在交互模式下可勾选/取消，默认全选
- 未指定平台时弹出平台多选，默认全选
- 选完后输出已选 skills 列表

## 发布 / 部署

- 不涉及 npm 包或线上部署

## 影响范围 / 风险

- Breaking change? 否
- 回滚方式：移除交互选择逻辑，恢复原行为
